# 知识库同步配置指南

## Phase 1: Git 多端同步

### 1.1 安装 Obsidian Git 插件

在 Obsidian 中：
1. 设置 → 社区插件 → 搜索「Obsidian Git」
2. 安装并启用

### 1.2 配置 Git 仓库

插件设置：
- **Remote repository**: `https://github.com/你的用户名/仓库名.git`
- **Branch**: `main`
- **Auto commit**: ✓
- **Auto push**: ✓（建议 5 分钟）
- **Auto pull**: ✓

### 1.3 首次推送

```bash
cd E:\database\En_V
git add .
git commit -m "Initial commit: 悦年轻知识库"
git remote add origin https://github.com/你的用户名/仓库名.git
git push -u origin main
```

## Phase 2: 飞书同步

### 2.1 创建飞书自建应用

1. 登录 [open.feishu.cn](https://open.feishu.cn)
2. 创建自建应用 → 获取 `App ID` 和 `App Secret`
3. 开通权限：
   - `docx:document:create` — 创建文档
   - `drive:file:create_folder` — 创建知识库
   - `drive:media:upload` — 上传图片

### 2.2 配置同步脚本

编辑 `scripts/config.js`:

```javascript
feishu: {
  enabled: true,
  appId: '你的AppID',
  appSecret: '你的AppSecret',
  knowledgeSpace: {
    name: '悦年轻',
    description: '悦年轻知识库同步空间'
  }
}
```

### 2.3 运行同步

```bash
cd E:\database\En_V\scripts
npm install axios
node sync-feishu.js
```

## Phase 3: 火山引擎同步

### 3.1 开通火山引擎 API

1. 登录 [volcengine.com](https://volcengine.com)
2. 开通「内容管理」服务
3. 获取 `Access Key` 和 `Secret Key`

### 3.2 配置同步脚本

编辑 `scripts/config.js`:

```javascript
volcengine: {
  enabled: true,
  accessKey: '你的AccessKey',
  secretKey: '你的SecretKey',
  region: 'cn-north-1',
  knowledgeSpace: {
    name: '悦年轻',
    description: '悦年轻知识库同步空间'
  }
}
```

## 定时任务（可选）

使用 Windows Task Scheduler 或 macOS Cron 定时运行同步脚本：

```bash
# 每小时同步一次
node E:\database\En_V\scripts\sync-feishu.js
node E:\database\En_V\scripts\sync-volcengine.js
```

---

## 冲突处理

当双向同步出现冲突时：
1. Git 会标记冲突文件
2. 脚本会生成 `scripts/logs/conflicts.json`
3. 人工确认后，手动合并并重新提交
